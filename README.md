# Rendering Engine (WIP)

A modern real-time rendering engine currently in active development. The project focues on physically based rendering techniques.

## Features
 - Real-Time Lighting (IBL + Phong)
 - glTF Model Loading
 - GPU Usage (with Vulkan)

## Tech Overview
 - API: Vulkan
 - Language: C++

## Work in Progress

As this project is still under active development, the following roadmap contains planned features:
 - Shadow Mapping
 - Multi-Threading
 - Frame Graph
 - Ray Tracing (DDGI)
 - Codebase Cleanup
